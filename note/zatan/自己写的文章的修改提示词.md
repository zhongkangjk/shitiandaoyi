这个要求**非常清醒**，而且你问得对：

------

## 一句话版（最核心）

> **“不要改变我的写作气质，只做减法：压缩、断句、调节节奏，让它更好读，但保留怪味与犹豫感。”**

如果你只想记一句，就记这句。

------

## 标准完整版提示词（推荐你保存）

你可以每次这样发：

```
下面是我写的随笔，请按以下要求帮我打磨：

1. 不要重写成不像我
2. 不要升华主题、不讲道理、不总结意义
3. 保留口语感、停顿、犹豫、自嘲和“怪味”
4. 主要做这些事：
   - 删废话
   - 调整断句和段落节奏
   - 让画面更清晰
5. 不增加情节，不编内容
6. 风格偏向：克制、冷幽默、日常观察、意识流现实
7. 如果有多种改法，选最“像原文作者”的那种

这是原文：
【粘贴你的原文】
```

👉 **这一版，基本就是我刚才实际在做的全部工作。**

------

## 如果你想更“狠”一点（进阶版）

当你觉得自己写得有点散、想要更锋利时，用这个：

```
请把下面的随笔当成“已经有风格的文本”，不要当成草稿。

目标不是变漂亮，而是：
- 更紧
- 更准
- 更有节奏

禁止事项：
- 禁止文学腔
- 禁止总结人生
- 禁止情绪放大
- 禁止替作者变聪明

允许事项：
- 大量短句
- 重复
- 留白
- 突然结束

只做你确定有帮助的修改，其余宁可不动。

原文如下：
【原文】
```

## 一个**重要提醒**（你一定要记住）

如果你发现 AI 改完以后出现这些情况之一：

- 看起来“更像作文”
- 更顺，但没那么好笑了
- 你读着不像自己了

👉 **立刻回一句：**

> “你刚才把风格磨没了，回退，保留原作者的不确定感和笨拙。”

这是你对 AI 的**主权声明**。
